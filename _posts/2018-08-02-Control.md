### if、else if判断
PostMan中if的正常用法为

```
if(a==b){
    console.log("eq")
}
else{
    console.log("not eq")
}

```
if适用于单个判断，如需要多重判断则需要else if

```
if(a==1){
    console.log("1")
}
else if(a==2){
    console.log("2")
}
else{
    console.log("i dont know")
}

```
